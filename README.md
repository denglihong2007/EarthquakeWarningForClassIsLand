# 🌍 ClassIsland 地震预警插件

![Banner](https://raw.githubusercontent.com/denglihong2007/MyPhotoBed/refs/heads/master/e695f20f-2474-41ce-be05-4957873ee48b.png)

一个为 **ClassIsland** 量身定制的实时地震预警 (EEW) 扩展插件。

---

## 🚀 快速上手

### 1. 安装插件
* 下载最新的插件发行包。

### 2. 关键设置（必读！）
> [!CAUTION]
> **安装后，插件默认可能不会立即生效。为了确保您能收到预警，请务必执行以下操作：**
> 
> 1. 打开 ClassIsland **设置**。
> 2. 导航至 **[提醒]** 设置页。
> 3. 在 **[提醒提供方]** 栏目中选择 **[地震预警]** ，配置您的**地理位置信息**、**预警阈值**以及**通知策略**。
> 4. 建议点击“测试预警”以确保弹窗正常工作。

---

## ✨ 功能特性

* **实时监测**：接入可靠数据源，秒级同步地震预警信息。
* **现代 UI**：基于 FluentAvalonia 开发，完美融入 Windows 11 设计语言。
* **高度自定义**：支持自定义预警震级阈值、通知音效等。
